# Minecraft9 3D Prototype

This is the first playable 3D prototype for `Minecraft9`, built with `Ursina`.

## Current Features

- First-person movement
- Jumping and block collision
- Infinite chunk-style terrain loading around the player
- Left click to mine blocks
- Right click to place blocks
- `1` to `5` or mouse wheel to switch hotbar blocks
- Simple day and night sky color cycle
- Basic save file for player position and changed blocks

## Run

```bash
python source3d/main.py
```

## Save File

The prototype stores its save data in:

```text
source3d/world_save.json
```
